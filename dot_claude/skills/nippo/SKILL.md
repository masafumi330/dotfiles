# nippo — 日報管理スキル

ユーザーの入力: `$ARGUMENTS`

- **第1引数**（サブコマンド）: `init` / `check` / `add` / `memo` / `show` / `summary`
- **第2引数以降**（オプション）: タスク名やメモ本文など

第1引数を見てどのサブコマンドかを判断し、対応する処理を実行する。  
引数がない場合や不明な場合は使い方を表示する。

---

## 共通設定

- 日報ディレクトリ: `~/nippo/`
- ファイル名: `YYYY-MM-DD.md`（今日の日付は `date +%Y-%m-%d` で取得）
- 今日のファイルパス: `~/nippo/$(date +%Y-%m-%d).md`
- ディレクトリが存在しない場合は自動作成: `mkdir -p ~/nippo`

## テンプレート

See [template.md](template.md)

## `init` — 朝：今日の日報を作成

1. 今日のファイルが既に存在する場合は内容を表示して終了
2. `~/nippo/*.md` をソートして最新ファイルを特定（月曜・連休明けを考慮）
3. 最新ファイルの `# 明日やること` セクションを読み込み、引き継ぎタスクとして取得
4. 新規ファイルをテンプレートで作成し、引き継ぎタスクを `- [ ] <タスク名> [--:--]` 形式で `# 今日やること` に書き込む
5. 第2引数以降にタスク名が与えられていればそれも追記する（`- [ ] <タスク名> [--:--]`）
6. 追加タスクをインタラクティブに確認する:
   - 「他に今日やることはありますか？（なければそのままEnter）」
   - 入力があれば追記し、「他にありますか？」を繰り返す
7. 作成した日報を全文表示

**前日ファイルが存在しない場合（初回など）**: 引き継ぎなしで新規作成する

---

## `check` — タスク完了をチェック

1. 今日のファイルを読み込む（なければ「先に `/nippo init` を実行してください」）
2. 未完了タスク（`- [ ]`）を番号付きで一覧表示
3. 第2引数でタスク番号が与えられていればそれを使用。なければインタラクティブに選ばせる
4. 現在時刻を取得: `date +%H:%M`
5. 該当行を更新:
   - `- [ ] <タスク名> [--:--]` → `- [x] <タスク名> [HH:MM]`
6. 「備考・メモはありますか？（なければEnter）」と聞き、入力があれば `# 学び・気づき・悩み・メモ` に追記
7. 更新後の `# 今日やること` セクションを表示

---

## `add` — 割り込みタスクを追加

1. 今日のファイルを読み込む
2. タスク名を決定:
   - 第2引数以降が与えられていればそれをタスク名として使用（スペース結合）
   - なければ「追加するタスク名を入力してください」とインタラクティブに聞く
3. `# 今日やること` セクションの末尾に追記:

   ```
   - [ ] 差し込み: <タスク名> [--:--]
   ```

4. 更新後の `# 今日やること` セクションを表示

---

## `memo` — 学び・気づきをメモ

1. 今日のファイルを読み込む
2. メモ内容を決定:
   - 第2引数以降が与えられていればそれをメモとして使用（スペース結合）
   - なければ「メモ内容を入力してください」とインタラクティブに聞く
3. `# 学び・気づき・悩み・メモ` セクションの末尾に追記:

   ```
   - <メモ内容>
   ```

4. 「メモを追加しました」と伝え、セクションを表示

---

## `show` — 現在の日報を表示

1. 今日のファイルを読み込む（なければ「先に `/nippo init` を実行してください」）
2. ファイルの全内容をそのまま表示

---

## `summary` — 終業処理

1. 今日のファイルを読み込む
2. 完了したタスク一覧を時系列順に並べ替える
3. 完了時刻から、タスクにかかった時間を計算し、(--h--m)形式で書く
4. 未完了タスクを「明日やること」に移動
5. 「明日やることを追加しますか？（なければEnter）」と聞き、入力があれば追記
6. メモ欄の部分は、内容精査して、コンパクトに書き直す
7. 最終的な日報の全内容を表示して完了

---

## ファイル操作の原則

- 読み込み: Read ツールを使用
- 編集: Edit ツールで対象セクションの該当行のみ書き換える
- 時刻取得: `date +%H:%M`（Bashで実行）
- 最新ファイル検索: `ls ~/nippo/*.md 2>/dev/null | sort | tail -1`

## セクション操作

各セクションは `#` で始まる見出し行で区切られる。  
編集時は見出し行を変更せず、セクション内のコンテンツのみを操作する。
